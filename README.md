1. **Project Name**
  **Olympic Insight Dashboard | Power BI Data Analytics Project**

2. **Project Description**
  The Olympic Insight Dashboard is an interactive Power BI dashboard designed to analyze Olympic Games data across different countries, sports, athletes, and
  medal categories. It provides meaningful insights into participation trends, medal distribution, country performance, and sport-wise achievements through
  dynamic visualizations and filters.

    The dashboard enables users to explore Olympic performance over different years by using interactive slicers, KPI cards, charts, maps, and tables, making the       data easy to understand and analyze.

**3. Tech Stack**
The following tools and technologies were used to build this project:

  Power BI Desktop

  Power Query (Data Cleaning & Transformation)

  DAX (Data Analysis Expressions)

  Data Modeling

  Microsoft Excel / CSV Dataset

  Power BI Maps

  Git & GitHub

**4. Data Source**

Dataset
  
  Source: Kaggle

Dataset contains historical Olympic Games information including athlete participation, country, sport, season, event, gender, and medal details.

  Dataset Structure

The dataset includes columns such as:

  Athlete Name

  Country (NOC)

  Year

  Season

  Sport

  Event

  Sex

  Medal

  Gold Medal

  Silver Medal

  Bronze Medal

Each row represents one athlete's participation in an Olympic event.
The data was cleaned and transformed using Power Query before being loaded into Power BI.

**5. Business Problem**

Organizations, researchers, sports analysts, and Olympic enthusiasts often struggle to analyze large volumes of Olympic data.
Without visualization, it becomes difficult to answer questions such as:

  Which country has won the most medals?

  Which sport contributes the highest number of medals?

  How has Olympic participation changed over the years?

  Which athletes have won the highest number of medals?

  What are the overall Olympic participation statistics?

This dashboard solves these problems by presenting the data in an interactive and visually appealing format.

**6. Goal of the Dashboard**

The main objective of this dashboard is to:

Analyze Olympic participation across different years.

Compare country-wise medal performance.

Identify top-performing sports.

Monitor total medals and athlete participation.

Enable users to filter data dynamically.

Provide quick business insights using KPIs and interactive visuals.

**7. Dashboard Walkthrough**

KPI Cards

The dashboard begins with eight KPI cards that provide a quick overview of Olympic statistics.

Total Country

Displays the total number of participating countries.

Total Athletes

Shows the total number of athletes included in the dataset.

Total Events

Displays the total Olympic events.

Sports

Shows the total number of sports.

Total Medals

Displays the overall medals won.

Gold Medal

Shows total gold medals.

Silver Medal

Displays total silver medals.

Bronze Medal

Displays total bronze medals.


These KPI cards help users understand the overall Olympic statistics instantly.

**Filters (Slicers)**

Interactive slicers allow users to filter dashboard data by:

Year

Country

Gender

Sport

Season (Summer/Winter)

Medal Type

These filters make the dashboard dynamic and improve data exploration.
